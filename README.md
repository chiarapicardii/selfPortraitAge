# Self-Portrait Age: The Age Evolution of Self-Portraits

Project for the Information Visualization course, University of Bologna, a.y. 2025/2026.

## Repository structure
- `SelfPortraitsAverageAge.ipynb` — full data pipeline: SPARQL queries (ArtResearch + Wikidata), cleaning, exploratory analysis, and iterative chart design
- `slide/` — final take-home slide (image/HTML)
- `data/` — intermediate and cleaned datasets (CSV/JSON)

## Data sources
- [ArtResearch](https://artresearch.net/) SPARQL endpoint (CIDOC-CRM / Pathos data model)
- [Wikidata](https://www.wikidata.org/) SPARQL endpoint (artist biographical data)

## Interpretation
Self-portraiture remained a predominantly youthful practice, but the average age of self-portrait production rose steadily after 1850 coinciding with the emergence of modern art movements.
