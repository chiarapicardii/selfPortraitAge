# Self-Portrait Age: The Age Evolution of Self-Portraits

Project for the Information Visualization course, University of Bologna, a.y. 2025/2026.

## Repository structure
- `SelfPortraitsAverageAge.ipynb`- full data pipeline and documentation.
- `slide/`- final slide.
- `data/`- intermediate and cleaned datasets.

## Data sources
- [ArtResearch](https://artresearch.net/) SPARQL endpoint
- [Wikidata](https://www.wikidata.org/) SPARQL endpoint
